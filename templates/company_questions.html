🔍 Google Interview Questions

▼ Two Sum - Find two numbers that add up to target Easy

Problem: Given an array of integers, find two numbers that add up to a specific target number.

Approach: Use a hash map to store seen numbers. For each number, check if (target - number) exists in the map.

Time Complexity: O(n) | Space Complexity: O(n)

▼ Longest Substring Without Repeating Characters Medium

Problem: Find the length of the longest substring without repeating characters.

Approach: Use sliding window with a hash map to track character indices. Expand the window, contract when duplicate found.

Time Complexity: O(n) | Space Complexity: O(min(m, n))

▼ Design a URL Shortening Service (System Design) Hard

Problem: Design a service like bit.ly that shortens long URLs.

Key Components: URL mapping database, hash function for encoding, rate limiting, CDN for distribution.

Considerations: Scalability, collision handling, expiration policies, analytics.

▼ Merge K Sorted Lists Hard

Problem: Merge K sorted linked lists into one sorted linked list.

Approach: Use a min-heap to efficiently get the smallest element from K lists at each step.

Time Complexity: O(n*k*log(k)) | Space Complexity: O(k)

💻 Microsoft Interview Questions

▼ Reverse a String Easy

Problem: Reverse a given string in-place.

Approach: Use two pointers at start and end, swap characters and move pointers towards center.

Time Complexity: O(n) | Space Complexity: O(1)

▼ Binary Tree Level Order Traversal Medium

Problem: Return the level order traversal (BFS) of a binary tree.

Approach: Use a queue to traverse level by level. Add all children of current level before moving to next.

Time Complexity: O(n) | Space Complexity: O(w) where w is max width

▼ Word Ladder (Graph Problem) Hard

Problem: Find shortest path from start word to end word, changing one letter at a time.

Approach: BFS where each neighbor is a word with one letter different. Use word patterns for efficiency.

Time Complexity: O(n*L²) | Space Complexity: O(n)

▼ Serialize and Deserialize Binary Tree Hard

Problem: Design an algorithm to serialize and deserialize a binary tree.

Approach: Use pre-order traversal with special markers for null nodes. Deserialize by building tree back from sequence.

Time Complexity: O(n) | Space Complexity: O(n)

🔶 Amazon Interview Questions

▼ Climbing Stairs - Dynamic Programming Easy

Problem: Climb n stairs. Each step you can climb 1 or 2 stairs. Count distinct ways.

Approach: DP where dp[i] = dp[i-1] + dp[i-2]. Also known as Fibonacci sequence.

Time Complexity: O(n) | Space Complexity: O(n) or O(1) with optimization

▼ Course Schedule (Topological Sort) Medium

Problem: Given prerequisites, determine if you can complete all courses.

Approach: Build graph and detect cycle using DFS or Kahn's algorithm for topological sort.

Time Complexity: O(V + E) | Space Complexity: O(V + E)

▼ Number of Islands (Graph DFS/BFS) Medium

Problem: Given a grid, count the number of islands (connected land cells).

Approach: DFS/BFS from each unvisited land cell to mark entire island, increment counter.

Time Complexity: O(m*n) | Space Complexity: O(m*n)

▼ Design a Movie Rental System (LLD) Hard

Problem: Design classes for a movie rental system with rentals and returns.

Approach: Movie, Customer, Rental, Store classes. Handle stock management and pricing.

Key Points: OOP principles, encapsulation, polymorphism for different movie types.
