# AI Smart Interview Coach

## Overview

AI Smart Interview Coach is a web-based application developed using Python and Flask that helps users prepare for technical interviews through mock interviews, learning modules, and performance analysis.

## Features

* User Registration and Login
* Technical Interview Questions
* Learning Modules
* Mock Interview Sessions
* Performance Evaluation
* Dashboard and Result Tracking
* SQLite Database Integration

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* SQLite

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Gnapika15/AI-SMART-INTERVIEW-COACH.git
```

2. Navigate to the project folder:

```bash
cd AI-SMART-INTERVIEW-COACH
```

3. Install dependencies:

```bash
pip install flask
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000/
```
